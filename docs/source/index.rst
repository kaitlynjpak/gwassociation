gwassociation documentation
===========================

``gwassociation`` is a Python package for evaluating associations between
gravity-wave sky maps and electromagnetic transient candidates.  It provides a
small high-level API, command-line tools, and lower-level numerical utilities for
sky, distance, radial, and temporal overlap calculations.

Start with :doc:`usage` for installation and workflow examples, then use
:doc:`api` for module-level reference documentation.
